Como utilizar a API

db.sql:
- Copie o script do banco de dados para um ambiente MySQL e execute os comandos de criação deo banco, suas tabelas e triggers.

app.js:
- Na linha 13, insira a senha de acesso do seu banco de dados.

Após os passos anteriores, execute o comando "node app.js" no terminal, assim, iniciando o servidor da api no http://localhost:8081
